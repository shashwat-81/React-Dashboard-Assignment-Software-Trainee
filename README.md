# React Dashboard Assignment

## Run locally
1. npm install  
2. npm run dev  
3. Open the URL printed by Vite (usually http://localhost:5173).

## Features
- Add / edit / remove widgets  
- Move widgets between categories  
- Add categories  
- Global search  
- localStorage persistence
